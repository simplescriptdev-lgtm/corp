Що всередині:

 - Історії з Редагувати/Видалити для кожного розділу.
 - Єдиний endpoint дій: insurance.php (обробляє UAH/FX/власник/операційний/IT/благодійний/рух капіталу).

Щоб працювали суми "надходжень" у розділах:
 - Додавайте надходження через вкладку "Рух капіталу".
 - Поле "Куди зарахувати": owner / operational / it / charity / insurance
 - Джерело: owner (власник) або bank (банк)
 - Далі модулі беруть агрегати з таблиці capital_inflows за категорією.
